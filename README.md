<h2 id="application">Como rodar a aplicação ▶️</h2>

No terminal, clone o projeto:

```bash
git clone https://github.com/MiguelLuan/Adivinhar-numero.git
```

Abra a pasta do projeto:

```bash
cd  Adivinhar-numero
```

Agora abra o editor de código:

```bash
code .
```
